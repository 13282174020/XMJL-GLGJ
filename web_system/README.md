# 智能文档生成系统 - AI 增强版

基于 Flask 的 Web 应用，支持上传需求文档，调用阿里云百炼平台 AI 自动生成可行性研究报告等多种文档。

## ✨ 核心功能

### 1. 🤖 AI 智能生成
- 调用阿里云百炼平台 API 生成定制化内容
- 支持多种大模型：通义千问、GLM、Kimi、MiniMax
- 根据上传的需求文档智能分析
- 支持用户补充额外要求

### 2. 🔑 API Key 配置
- 在线验证 API Key 有效性
- 支持临时输入或保存配置
- 无 API Key 时使用模板生成

### 3. 📄 文档智能解析
- 支持 Word (.docx/.doc)、TXT 格式
- 自动提取文档文本内容
- 拖拽上传，便捷操作

### 4. 📋 多模板支持
| 模板类型 | 说明 | 章节数 |
|---------|------|--------|
| 未来社区可行性研究报告 | 适用于未来社区建设项目 | 15 章 |
| 通用项目可行性研究报告 | 适用于各类项目立项 | 7 章 |
| 智慧社区建设方案 | 适用于智慧化改造项目 | 7 章 |
| 商业计划书 | 适用于创业融资、商业规划 | 8 章 |
| 项目建议书 | 适用于项目立项申报 | 10 章 |
| 资金申请报告 | 适用于政府资金申请 | 8 章 |
| 社会稳定风险评估报告 | 适用于重大项目稳评 | 6 章 |

### 5. 🎯 多模型支持
| 模型 | 提供商 | 特点 |
|------|--------|------|
| 通义千问-Max/Plus/Turbo | 阿里云 | 长文本生成能力强 |
| GLM-4/3-Turbo | 智谱 AI | 逻辑推理能力强 |
| Kimi | 月之暗面 | 擅长长上下文理解 |
| MiniMax | MiniMax | 通用大模型，性价比高 |

## 🚀 快速开始

### 安装依赖
```bash
pip install -r requirements.txt
```

### 启动服务
```bash
python app.py
```

或双击运行 `启动服务.bat`

### 访问系统
浏览器打开：http://localhost:5000

## 📖 使用说明

### 基本流程
1. **上传文件**（可选）
   - 模板文件：参考格式用
   - 需求文档：内容来源

2. **选择报告类型**
   - 未来社区可行性研究报告
   - 通用项目可行性研究报告
   - 智慧社区建设方案
   - 商业计划书
   - 项目建议书
   - 资金申请报告
   - 社会稳定风险评估报告

3. **选择 AI 模型**（使用 AI 生成需要）
   - 通义千问-Max/Plus/Turbo（阿里云）
   - GLM-4/3-Turbo（智谱 AI）
   - Kimi（月之暗面）
   - MiniMax

4. **配置 API Key**（使用 AI 生成需要）
   - 阿里云百炼：访问 [阿里云 DashScope 控制台](https://dashscope.console.aliyun.com/)
   - 智谱 AI：访问 [智谱 AI 开放平台](https://open.bigmodel.cn/)
   - 月之暗面：访问 [Kimi 开放平台](https://platform.moonshot.cn/)
   - MiniMax：访问 [MiniMax 开放平台](https://platform.minimaxi.com/)
   - 点击"验证"按钮确认有效

5. **填写生成要求**（可选）
   - 重点关注的场景
   - 特殊要求说明
   - 投资限额等

6. **生成并下载**
   - 点击"开始生成报告"
   - 等待生成完成
   - 下载 Word 文档

### API Key 获取步骤

#### 阿里云百炼（通义千问系列）
1. 访问 https://dashscope.console.aliyun.com/
2. 注册/登录阿里云账号
3. 开通 DashScope 服务
4. 在 API Key 管理页面创建/查看
5. 复制 API Key 到系统使用

#### 智谱 AI（GLM 系列）
1. 访问 https://open.bigmodel.cn/
2. 注册/登录账号
3. 在 API Key 管理页面创建 API Key
4. 复制 API Key 到系统使用

#### 月之暗面（Kimi）
1. 访问 https://platform.moonshot.cn/
2. 注册/登录账号
3. 在控制台获取 API Key
4. 复制 API Key 到系统使用

#### MiniMax
1. 访问 https://platform.minimaxi.com/
2. 注册/登录账号
3. 在 API Key 管理页面创建 API Key
4. 复制 API Key 到系统使用

## 🔌 API 接口

### POST /api/generate
生成文档

**参数：**
- `template_type`: 报告类型
- `template`: 模板文件（可选）
- `requirement_file`: 需求文件（可选）
- `requirement`: 文本要求（可选）
- `api_key`: Qwen API Key

### POST /api/validate-key
验证 API Key 有效性

### GET /api/download/<filename>
下载生成的文档

### POST /api/cleanup
清理过期文件

## 💡 使用技巧

1. **有需求文档时**：上传文档，AI 会基于文档内容生成
2. **无需求文档时**：在"生成要求"中详细描述需求
3. **不使用 AI**：留空 API Key，使用预设模板生成
4. **批量生成**：修改参数后可多次生成不同版本

## ⚠️ 注意事项

1. API Key 会消耗阿里云账户额度，请合理使用
2. 上传的文件和生成的文档保留 1 小时后自动清理
3. 建议不要上传包含敏感信息的文档
4. AI 生成内容可能需要人工审核校对

## 🔧 自定义扩展

### 添加新模板
在 `app.py` 的 `TEMPLATE_TYPES` 字典中添加：

```python
'new_template': {
    'name': '新模板名称',
    'chapters': [
        ('第 1 章 标题', ['1.1 节标题', '1.2 节标题']),
        # ...
    ]
}
```

### 修改 AI 模型
在 `call_qwen_api()` 函数中修改 `model` 参数：
- `qwen-max`: 最强模型
- `qwen-plus`: 平衡模型
- `qwen-turbo`: 快速模型
