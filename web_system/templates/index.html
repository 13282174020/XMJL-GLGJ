🔑 如何获取 API Key

  1. 访问 阿里云 DashScope 控制台
  2. 注册/登录阿里云账号
  3. 开通 DashScope 服务
  4. 在 API Key 管理页面创建/查看 API Key
  5. 复制 API Key 到本系统使用

📝 使用流程

  1. 上传需求文档（Word/TXT 格式）
  2. 选择报告类型（未来社区/通用项目/智慧社区）
  3. 输入 API Key（使用 AI 生成必需）
  4. 填写额外要求（可选）
  5. 点击生成，等待完成
  6. 下载生成的 Word 文档
